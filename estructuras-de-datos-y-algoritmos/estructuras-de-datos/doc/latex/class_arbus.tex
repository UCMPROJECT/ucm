\subsubsection*{template$<$class Clave, class Valor$>$class Arbus$<$ Clave, Valor $>$}

\-Implementación dinámica del \-T\-A\-D \hyperlink{class_arbus}{\-Arbus} utilizando nodos con un puntero al hijo izquierdo y otro al hijo derecho.

\-Las operaciones son\-:


\begin{DoxyItemize}
\item \-Arbus\-Vacio\-: operación generadora que construye un árbol de búsqueda vacío.
\end{DoxyItemize}


\begin{DoxyItemize}
\item \-Inserta(clave, valor)\-: generadora que añade una nueva pareja (clave, valor) al árbol. \-Si la clave ya estaba se sustituye el valor.
\end{DoxyItemize}


\begin{DoxyItemize}
\item borra(clave)\-: operación modificadora. \-Elimina la clave del árbol de búsqueda. \-Si la clave no está, la operación no tiene efecto.
\end{DoxyItemize}


\begin{DoxyItemize}
\item consulta(clave)\-: operación observadora que devuelve el valor asociado a una clave. \-Es un error preguntar por una clave que no existe.
\end{DoxyItemize}


\begin{DoxyItemize}
\item esta(clave)\-: operación observadora. \-Sirve para averiguar si se ha introducido una clave en el árbol.
\end{DoxyItemize}


\begin{DoxyItemize}
\item \hyperlink{class_arbus_a01b9f1970a30516fe327d7308d70a0b9}{es\-Vacio()}\-: operacion observadora que indica si el árbol de búsqueda tiene alguna clave introducida.
\end{DoxyItemize}
