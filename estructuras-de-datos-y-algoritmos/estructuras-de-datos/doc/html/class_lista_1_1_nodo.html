Detailed Description

template<class T>
class Lista< T >::Nodo

Clase nodo que almacena internamente el elemento (de tipo T), y dos punteros, uno al nodo anterior y otro al nodo siguiente. Ambos punteros podrían ser NULL si el nodo es el primero y/o último de la lista enlazada.
